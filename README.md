# 🧲 Item Magnet Mod (Minecraft 1.21.11)

Eine kleine, aber extrem nützliche Utility-Mod für Minecraft, die dir das Einsammeln von Items abnimmt! Perfekt für lange Mining-Touren oder das automatische Farmen von Ressourcen.

---

## ✨ Features

* **Automatische Anziehung:** Sobald du den Magneten hältst, zieht er alle am Boden liegenden Items im Umkreis von **6 Blöcken** magisch zu dir heran.
* **Haupthand & Offhand:** Der Magnet funktioniert flexibel – du kannst ihn in der Haupthand halten oder in deine Nebenhand (Schild-Slot) packen, um gleichzeitig dein Werkzeug zu benutzen.
* **Vanilla-Look:** Das Item kommt mit einer eigens erstellen 16x16 PixelArt-Textur, die sich nahtlos in das originale Design von Minecraft einfügt.
* **Optimierte Performance:** Dank modernem Forge-Code (1.21.11) werden Item-Bewegungen flüssig und server-schonend synchronisiert.

## 📦 Installation

1. Stelle sicher, dass du **Forge für die Version 1.21.11** installiert hast.
2. Navigiere zu den **Releases** hier auf GitHub und lade dir die aktuellste `.jar`-Datei herunter.
3. Drücke `Win + R`, tippe `%appdata%/.minecraft` ein und öffne den Ordner.
4. Schiebe die heruntergeladene `.jar`-Datei in den dortigen **`mods`**-Ordner (falls er nicht existiert, erstelle ihn einfach).
5. Starte Minecraft über das Forge-Profil – fertig!

---

## 💡 Cheat-Befehl (Kreativmodus)
Falls du den Magneten im Kreativmodus direkt per Chat-Befehl erhalten möchtest, nutze diesen Command:
```text

```
